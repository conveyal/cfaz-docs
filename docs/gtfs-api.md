# GTFS API Documentation

The GTFS API allows for structured access via GraphQL to the General Transit Feed Specification (GTFS) data for all transit providers included in CarFreeAtoZ.

The GTFS API GraphQL endpoint is located at [https://gtfs.conveyal.com/api/manager/graphql](https://gtfs.conveyal.com/api/manager/graphql).

## GTFS API Queries

For basic information on query structure, see the [GTFS API documentation](https://github.com/conveyal/gtfs-api/#the-basics).
